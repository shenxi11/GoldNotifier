# GoldNotifier 移动端深色行情 UI v1.3 落地说明

## 业务目标

根据 `设计文档/goldpriceapp.tsx` 的前端设计稿，将 Android 首页调整为深色行情盘风格，同时保留现有实时行情、趋势图、通知栏显示和刷新控制能力。

## 已落地行为

- 首页整体改为黑色背景和深灰分段面板，贴近设计稿的行情终端视觉。
- 顶部栏展示 `现货黄金 XAU`、数据源、单位和行情状态，不新增无实际业务功能的搜索、铃铛或返回入口。
- 主行情区突出展示当前价格、涨跌额/涨跌幅、更新时间和实时状态点。
- OHLC 指标改为一行四列：今开、昨收、最高、最低。
- 趋势区采用深色面板，时间尺度选择改为紧凑分段控件，右侧展示趋势摘要。
- 趋势图保留服务端历史点、多时间尺度和左侧动态价格刻度能力，图表高度调整为更接近设计稿的行情图比例。
- 设置区调整为深色块，保留状态栏显示、无障碍权限入口、刷新频率和立即刷新。
- 底部继续展示权限说明和行情风险提示。

## 当前边界

- 本轮只做 Android 客户端 UI 重设计，不改服务端接口、数据模型、刷新策略或 Release API 地址。
- 设计稿中的 React、Recharts、lucide 图标不引入 Android 项目。
- Release 包仍保持原有正式地址配置；本轮上线目标是提交并推送客户端仓库源码。
